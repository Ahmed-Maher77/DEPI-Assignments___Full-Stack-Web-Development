About DEPI

The Digital Egypt Pioneers Initiative (DEPI) is a program designed to help students build foundational and advanced skills in web development. It introduces technologies such as HTML, CSS, JavaScript, Git, and more through weekly hands-on tasks.

Objectives

  • Build real-world web applications using HTML, CSS, and JavaScript.
  • Understand layout principles and responsive design.
  • Use version control with Git and GitHub.
  • Develop strong problem-solving and debugging skills.

Weekly Tasks

  1. Week 1: Introduction to HTML structure and basic tags.
  2. Week 2: Styling with CSS and building layouts.
  3. Week 3: Responsive design and media queries.
  4. Week 4: Header and Navigation Bar (this task).
  5. Week 5: JavaScript basics and DOM manipulation.
